# ✈️ FlightFight – Flight Booking Application

FlightFight is a full-stack flight booking web application that allows users to search, book, and manage flights through an easy-to-use interface.

---

## 🚀 Features

- User Authentication (Login / Register)
- Search available flights
- Book flights
- View booking details
- Admin flight management (optional)
- Responsive UI
- REST API based backend

---

## 🛠️ Tech Stack

### Frontend
- React.js (Client)
- CSS / Custom Styling
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB (MongoDB Compass / Atlas)
